# Requirements

```bash
pacman -S ansible
```

# Install

```bash
git clone --recurse-submodules -j8 git@github.com:mrdev023/ansible_perso_arch.git
```

```bash
cd ansible_perso_arch
```

```bash
ansible-playbook -i inventory/localhost playbook.yml
```
